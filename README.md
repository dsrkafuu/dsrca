## DSRCA

Monorepo for DSR projects.

### Packages

- `@dsrca/cdn`
- `@dsrca/config`
- `@dsrca/design`

### Development

```
npx lerna run dev
npx lerna version
npx lerna run build
npx lerna publish from-package
```
